# プロジェクト記憶: yutai-manager / Project Memory: yutai-manager

## 1. 基本理解 / Basic Understanding
* **ゴール / Goal**: 株主優待情報を管理・可視化するWebアプリケーションの開発
* **成果物 / Deliverables**: 
  - ユーザーが保有銘柄を登録できるUI
